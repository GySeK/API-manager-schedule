Manager Schedule API
============

Эксель для зуммеров

## Требуемый софт
- [Postgresql](https://www.postgresql.org/)
## Преднастройка

```bash
$ npm install
```
Зайди в файл .env.local или .env и отредактируй для себя и не забудь создать БД с названием из конфига 

## Запуск проекта

```bash
# разработка
$ npm run start

# крутая разработка
$ npm run start:dev

# прод
$ npm run start:prod
```

## Тесты

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
